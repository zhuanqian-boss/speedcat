## 项目介绍

我们是一群身处海外、仍在追逐理想的普通人。深知无论是出国学习、工作还是交流，稳定顺畅的网络都是刚需。正因如此，我们创建了 **SpeedCAT 闪电猫**，希望为更多用户提供可靠、好用的机场服务体验。

闪电猫属于 **V2Ray 架构机场**，底层采用 SS 协议，兼容市面上绝大多数第三方客户端。当前已部署 **120+ 全球节点**，单节点峰值速度可达 **5Gbps**，流畅播放 4K 视频毫无压力。

我们不追求花里胡哨的噱头，而是专注于“稳定”二字。因为真正用过的人都明白，网络中断有多让人崩溃。闪电猫会持续运营下去，也希望能得到大家的支持与陪伴。
